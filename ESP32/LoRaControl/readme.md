# LoRaControl -made by Eu93n3-
## 概要
ES920LR, ESP32用

LoRaから命令を受け取り水門制御や水位計測を行うプログラム

## 使い方

| 接続するもの                 | ESP32側 |
|:-----------------------------|---------|
| ES920LRの 8番(UART_TX)       | 16番(RX)|
| ES920LRの 9番(UART_RX)       | 17番(TX)|
| ES920LRの24番(RESET)         | 14番    |
| LED                          |  4番    |
| 水位センサの出力ピン         | 33番    |
| モーターの回転方向指定用ピン | 13番    |
| モーターの回転制御用ピン     | 12番    |

GNDなどは適宜接続すること
1. ピンを上記のように接続
2. ESP32に書き込む
3. あとは勝手にやってくれるはず
* シリアルモニタからの入力も受け付ける...はず。その辺の使い方はLoRaConsole参照
